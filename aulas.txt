Anatomia das Classes

Sintaxe de declaração de uma nova classe: 
public class MinhaClass {
    // O meu código

!! Usar nomes intuitivos !!

public class MinhaClasse{
    // Classe executável -> Tem capacidade de inicializar o programa, tem de ter um método especial - main
    public static void main (String [] args){
        // Corpo da classe
        System.out.println("Olá a todos!"); // Classe system, dá-me um output, printando
    }
}

- Todas as classes precisam de existir dentro do diretório src do meu projeto
- Se houver palavra composta cada letra inicial precisa ser maiúscula, ex:. MinhaClasse

Padrão de nomenclatura

Arquivo .java -> Todos os arquivos .java devem começar por letra Maiúscula. Ex: Calculadora.java, CalculadoraGrafica.java

Nome da classe no arquivo -> Deve ter o mesmo nome do arquivo. Ex: Calculadora.java, public class Calculadora {}

Nome da variável -> escritas com letra minúscula, mas se a palavra for composta devemos fazer: anoFabricacao (camelCase)
    -> String BR = "Brasil";
        BR = "Brazil"; // Não devemos fazer isto se quisermos manter o valor de BR sempre igual, para isso usamos maiúscula e usamos:
        final String BR = "Brasil"; // Valor imutável

    -> Para os nomes, usar apenas letras, _, $ ou números de 0-9
    -> Nunca começar com número
    -> Iniciar com letra minúscula e seguir camelCase
    -> Não incluir espaços
    -> Nome único dentro de um escopo

Tipos e variáveis

Tipos de dados:
- No java existem palavras reservadas para representar os tipos de dados básicos que precisam ser manipulados para construir programas
- Estes tipos de dados são conhecidos como tipos primitivos (primitive types):
    - byte (1 byte), short (2 bytes), int (4 bytes), long (8 bytes), *partes fracionárias:* float (4 bytes), double (8 bytes) - garantia de precisões, boolean, char
    - Não são objetovs, são valores brutos
    - Armazenados diretamente na pilha de memória (memory stack)

Declaração de variáveis
- Variável é a identificação de um espaço em memória utilizado pelo nosso programa
- A variável é composta por: tipo de dados + identificação + valor atribuído
- Um double é com . para decimal: double salarioMinimo = 2500.33;
- Long: long cpf 98765432109L
- float pi = 3.14F  -> Para perceber que é um float e não um double
- Java é fortemente tipado!!!!
-short numeroCurto = 1;
    int numeroNormal = numeroCurto;
    // short numeroCurto2 = numeroNormal; -> Não iria funcionar, pois o número normal é um int e o número curto é um short, e o Java não tem como adivinhar se o número normal vai caber dentro do número curto. Para resolver isso, precisamos fazer um cast, ou seja, dizer para o Java que temos a certeza que o número normal vai caber dentro do número curto. 
    short numeroCurto2 = (short) numeroNormal;

Variáveis e constantes
- Variável é uma área de memória, associada a um nome, que pode armazenar valores de um determinado tipo
- O tipo de dado define um conjunto de valores e um conjunto de operações
- Java é uma linguagem com muita rigidez de tipos
- Utilizamos uma referência (ponteiro) a um valor de memória, e esta pode ser redirecionada a outro valor (daí o nome "variável")

- Constantes são valores armazenados em memória, que não podem ser modificados depois de serem declarados
- São representadas pela palavra "final"
- São escritas em maiúsculas
- Exemplo: final double VALOR_DE_PI = 3.14;

Operadores
- Símbolos especiais que usamos para expressões/operações matemáticas
- Um programa é um conjunto de instruções lógicas que, quando executadas, produzem algum Resultado

Classificação de Operadores
- Atribuição -> símbolo de igualdade = ; é usado para definir valor inicial ou sobreescrever o valor de uma variável

Aritméticos
- Usado para realizar operações matemáticas entre valores numéricos
- + adição, - subtração, * multiplicação, / divisão
- a + quando usada em variáveis do tipo texto, procede à concatenação de texto em vez de adição

Unários
- São aplicados junto com outro operador Aritméticos
- Incrementam, decrementam, invertem valores numéricos e booleanos
- + unário de valor positivo
- - unário de valor negativo
- ++ unário de incremento de valor
- -- unário de decremento de valor
- ! unário lógico de negação

Ternário
- Operador de condição ternária é uma forma resumida para definir uma condição e escolher entre 2 valores
- É como um if normal mas escrito numa só linha
- É representado pelo símbolo ?
- <Expressão condicional> ? <Caso a condição seja true> : <Caso seja false>
- Substitui isto:
    int a,b

    a=5;
    b=6;

    String resultado = "";

    if(a==b)
        resultado = "verdadeiro";
    else 
        resultado = "falso";
    
    System.out.println(resultado);
- Por isto:
    int a,b

    a=5;
    b=6;

    String resultado = a==b ? "verdadeiro" : "falso";

- Com objetos utilizamos o equals em vez do ==. Compara conteúdos; só devemos usar o == com números
